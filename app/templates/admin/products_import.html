Column reference
  • product_slug: lookup key (auto-generated if empty for new products)
  • product_name: required for new products
  • price: required for new products; blank leaves unchanged on update
  • color_name: if present, the row defines a variant on the product
  • variant_sku: lookup key for existing variants on the product
  • stock_quantity: used when there's no color; otherwise use variant_stock_quantity
  • Booleans: TRUE/FALSE, 1/0, yes/no all accepted
